## epic.optargs
  
  js function analyser

## Installation

```bash
$ npm install epic.jsa
```


## Usage

```js
var analyser = require('epic.jsa');
```

```js

var test = function(_obj, id, _fn){ console.log(arguments); };

console.log(analyser(test));
```
